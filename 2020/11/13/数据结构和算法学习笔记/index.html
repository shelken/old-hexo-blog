HashMap 与 ConcurrentMap 和 LinkedHashMap 之间区别

  • HashMap 与 LinkedHashMap 都不是线程安全的，ConcurrentMap 是

  • LinkedHashMap在结构上是双向链表，因此可以双向遍历。且他是保证插入顺序的。HashMap 不保证插入顺序。

  • HashMap 比 ConcurrentMap 会快

  1. 哪种数据结构用于用Java实现HashMap？即使HashMap表示哈希表，也可以在JDK中使用数组和链表数据结构在内部实现。数组数据结构用作存储桶，而链表用于存储所有位于同一存储桶中的映射。从Java 8开始，一旦链表中的许多元素超过特定阈值以提高性能，链表就会被二进制搜索树动态替换。

计算二叉树中的节点

// 定义：count(root) 返回以 root 为根的树有多少节点
int count(TreeNode root) {
    // base case
    if (root == null) return 0;
    // 自己加上子树的节点数就是整棵树的节点数
    return 1 + count(root.left) + count(root.right);
}

void sort(int[] nums, int lo, int hi) {
    /****** 前序遍历位置 ******/
    // 通过交换元素构建分界点 p
    int p = partition(nums, lo, hi);
    /************************/


    sort(nums, lo, p - 1);

}

void sort(int[] nums, int lo, int hi) {

### 填充二叉树节点的右侧指针

- 重点是左节点的右节点与右节点的左节点如何相连：

  

![a](https://i.loli.net/2020/11/13/UYC6x1jeNJiDtvu.png)



```java
void connectTwoNode(Node node1, Node node2) {
    if (node1 == null || node2 == null) {
        return;
    }
    /**** 前序遍历位置 ****/
    // 将传入的两个节点连接
    node1.next = node2;

    // 连接相同父节点的两个子节点
    connectTwoNode(node1.left, node1.right);
    connectTwoNode(node2.left, node2.right);
    // 连接跨越父节点的两个子节点
    connectTwoNode(node1.right, node2.left);
}

void** **flatten**(TreeNode root) {
  *// base case*
  **if** (root == **null**) **return**;

  flatten(root.left);
  flatten(root.right);

  */**** 后序遍历位置 ****/*
  *// 1、左右子树已经被拉平成一条链表*
  TreeNode left = root.left;
  TreeNode right = root.right;

  *// 2、将左子树作为右子树*
  root.left = **null**;
  root.right = left;

  *// 3、将原先的右子树接到当前右子树的末端*
  TreeNode p = root;
  **while** (p.right != **null**) {
    p = p.right;
  }
  p.right = right;
}
